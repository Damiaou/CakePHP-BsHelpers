Installation

Vous avez deux choix d'installation pour ce plugin

  • Par Composer
  • Par copier/coller

Il est préférable de le faire par Composer.

Composer

Ajouter simplement cette ligne dans votre composer.json puis faites un composer update

{
  "require":{
      "webandcow/bs_helpers": "1.1.3"
  }
}

Copier/coller

Copiez/collez tout le projet Github dans votre dossier app/Plugin/BsHelpers

Activation du plugin

Ajoutez cette ligne dans votre fichier app/Config/bootstrap.php

CakePlugin::load('BsHelpers');

ou si vous avez déjà d'autres plugins de chargés : 

CakePlugin::loadAll();

Acitvation des helpers

Enfin, ajoutez les helpers dans le fichier app/Controller/AppController.php

<?php

class AppController extends Controller {

  public $helpers = array('BsHelpers.Bs', 'BsHelpers.BsForm'));

}

Et voilà, vous pouvez commencer à utiliser le BsHelper et le BsFormHelper !
