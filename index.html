BsHelpers

Plugin CakePHP pour utiliser le Framework Twitter Bootstrap plus facilement

Voir sur Github Télécharger


Ecrire sa grille rapidement

Grâce à une fonction bien pensée, plus besoin d'écrire des classes à rallonge pour vos colonnes.

Garde la synthaxe CakePHP

Avec le BsFormHelper, vous pouvez garder vos formulaire d'origines. Les fonctions select(), radio() et checkbox() sont revisitées pour qu'elles correspondent mieux à vos besoins.

Une pensée UX

En plus de faciliter l'usage du framework, le BsHelper propose des fonctions comme confirm() qui vous automatisera l'expérience utilisateur pour vos modals.

De nouvelles fonctionnalités pour les formulaires

Construisez facilement des input-group ou des datepicker en passant seulement par une fonction !

Automatisation du javascript et css

Plus besoin de télécharger les helpers et en plus les ressources externes, le plugin fournit tout un paquet directement utilisable une fois installé.

Juste un complément

Ces helpers ne vous empêcheront absoluement pas d'utiliser le HtmlHelper et le FormHelper natif de CakePHP.
