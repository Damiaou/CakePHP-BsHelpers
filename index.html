BsHelpers

CakePHP plugin to use the framework Twitter Bootstrap v3 more easily.

Github Project Download


Write grid faster

Through a thinking function well, no need to write long classes anymore.

Keep CakePHP synthax

With BsFormHelper, you can keep your original forms. Select(), radio() and checkbox() functions are revisited to better suit your needs.

Think UX

In addition to facilitating the use of the framework, the BsHelper offers features like confirm() that will automate the user experience for your modals.

New features for forms

Easily build input-group or datepicker passing by a function !

Automation of javascript and css

No need to download helpers, and more external resources, the plugin provides a whole package directly usable once installed.

Just additional

These helpers won't prevent you from using the HtmlHelper and native CakePHP FormHelper.
